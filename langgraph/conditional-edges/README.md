## Conditional Edges in LangGraph

### Watch the video

[YouTube Video](https://youtu.be/wGN_xvB_d6Y)

### Run the app

- Open a terminal and run `python graph.py`
- Provide a prompt

### Tweaks

- Change the number of iterations and the quality threshold

  - `num_iterations = 5`
  - `quality_threshold = 950`

- Change the LLM model from OpenAI to Ollama or another model by channging the llm variable
